# css

### 선택자의 종류

+ 전체선택자 : *
+ 태그(타입) : 태그명
+ 그룹 : 태그명
+ 클래스 :  .class 속성값
+ 아이디 : #id 속성값
+ 자식 : 부모태그>자식태그
+ 자속 : 조성태그> 자속태그
+ 형제 :  윗태그 ~ 아래태그 or 윗태그 + 아래태그 
+ 속성 : [속성명]/ [속성명]/[속성명]/[속성명]

  inlinestyle(영역존재)태그와 blockstyle 태그 - (행끝까지) 로 나뉨
